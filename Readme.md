📌 Descrição do projeto

Este projeto consiste no desenvolvimento de um Álbum de Figurinhas digital, criado com o framework Vue.js.

O sistema simula um álbum de colecionáveis onde o usuário pode visualizar uma lista de figurinhas e controlar quais já foram coletadas. A aplicação foi desenvolvida com foco em organização de código, componentização e uso de reatividade do Vue.

O projeto tem como objetivo praticar conceitos fundamentais do desenvolvimento frontend moderno, como manipulação de estado, atualização dinâmica da interface e filtragem de dados.

⚙️ Funcionalidades

O sistema conta com as seguintes funcionalidades:

📋 Exibição de uma lista de figurinhas disponíveis
✅ Marcar e desmarcar figurinhas como “coletadas”
🔍 Sistema de busca por nome da figurinha
🎯 Filtro para visualizar:
Todas as figurinhas
Apenas figurinhas coletadas
Apenas figurinhas faltando
📊 Controle do estado do álbum em tempo real


🧠 Conceitos utilizados
Durante o desenvolvimento foram aplicados os seguintes conceitos do Vue.js:
ref() para gerenciamento de estado reativo
computed() para filtros dinâmicos
Manipulação de arrays (map, find, filter)
Componentização da interface
Separação de lógica com composables (useAlbum)
Atualização reativa da interface sem recarregar a página

🗂️ Estrutura do projeto (resumo)
useAlbum.ts/js → lógica principal do álbum
stickersData → base de dados das figurinhas
Componentes Vue → interface do usuário
Estado reativo → controle de coleção e filtros

🚀 Objetivo do projeto
O objetivo principal deste projeto é aplicar conhecimentos de desenvolvimento frontend com Vue.js, simulando um sistema real de gerenciamento de coleção, além de treinar boas práticas de organização de código e reatividade.
